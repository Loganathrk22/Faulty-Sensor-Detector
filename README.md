# Faulty-Sensor-Detector
The main objective of this project is to design, implement, and analyze a brute force approach to identify a faulty sensor among multiple sensors measuring the same physical quantity. 
• To represent sensor data as a collection of readings obtained from multiple sensors measuring the same parameter.
• To implement a brute force algorithm that compares each sensor reading with every other sensor reading.
• To identify the faulty sensor by detecting the one whose readings consistently deviate from the majority of sensors.
• To understand how pairwise comparison helps in determining consistency and inconsistency among sensor values.
• To analyze the time complexity of the brute force approach and observe that it results in O(n²) complexity due to repeated comparisons.
• To study the trade-offs between simplicity and efficiency in algorithm design using the brute force method.
• To visualize the comparison process using a simple interface for better conceptual understanding.
• To enhance logical thinking and problem-solving skills through the use of loops and conditional statements.
• To demonstrate how basic algorithms can be applied to real-world problems such as fault detection in sensor systems.
• To strengthen fundamental knowledge in algorithm design, data analysis, and computational complexity.
