Time Complexity O(n²)
Space Complexity O(n)
Total Comparisons —
Algorithm Type Brute Force

For n sensors, the algorithm performs n×(n-1) ordered comparisons (outer × inner loops), making it O(n²) in time. Each match count is stored in an array of size n — O(n) space.
